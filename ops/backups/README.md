# ops/backups — артефакты для отката

Сюда складываются снимки конфигураций (DNS, секреты-индексы, и т.п.) перед
рискованными изменениями. Содержимое — **не коммитить**, см. `.gitignore`.

## Cloudflare DNS

Файлы вида `cloudflare-pepperoni.tatar-<timestamp>.bind` и `.json` — полный
экспорт зоны на момент времени. Восстановление:

1. Cloudflare Dashboard → `pepperoni.tatar` → DNS → Records → **Import**.
2. Выбрать `.bind` файл нужного снимка.

JSON-снимок — для быстрого diff'а через `jq`, не для импорта.

## Когда снимать

- Перед выдачей DNS-Edit токена кому-либо (включая MCP-агентов).
- Перед массовой правкой записей вручную.
- После любого инцидента — фиксировать состояние «как было до восстановления».
