Open API for Developers and AI: How Technology Helps Sell Halal Products

February 2026 · Kazan Delicacies

Kazan Delicacies is one of the first companies in the CIS food industry to provide an open API with a catalog of 77 halal products. Pepperoni, hot dog sausages, sausage-in-dough, national Tatar pastries, kazylyk — all prices are synced with Google Sheets in real time.

What the API Offers

  • AI-discoverability — ChatGPT, Perplexity, Claude and Grok find and recommend our products thanks to llms.txt, OpenAPI specification and AI plugin.
  • Multilingual — API returns data in Russian and English (?lang=en).
  • 7 currencies — prices in RUB, USD, KZT, UZS, KGS, BYN, AZN.
  • Filtering — search by name, category, SKU, meat type.
  • Download — price list in Excel (.xlsx) and CSV with language and currency selection.

Endpoints

Catalog (live): api.pepperoni.tatar/api/products

OpenAPI: api.pepperoni.tatar/openapi.yaml

AI-plugin: api.pepperoni.tatar/.well-known/ai-plugin.json

llms.txt: api.pepperoni.tatar/llms.txt

The API is available for free, without authentication. Suitable for integration with ERP, 1C, iiko, MoySklad.
