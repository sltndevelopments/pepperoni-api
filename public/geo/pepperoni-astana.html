Да. Сертификат Духовного управления мусульман Республики Татарстан признан казахстанскими халяль-органами и сетями HoReCa. Как оформить поставку пепперони оптом из России в Астану?
Поставка осуществляется автотранспортом по маршруту Казань — Астана (≈ 2 500 км, 48–72 часа). Работаем на условиях EXW Казань и DAP Астана. Таможенное оформление в рамках ЕАЭС — без импортных пошлин, по ветеринарным свидетельствам РФ, которые мы оформляем на каждую партию.

Минимальный заказ для международной поставки — от 200 кг. Для первого сотрудничества возможна тестовая партия с согласованными условиями. Уточните детали у менеджера по телефону +7 987 217-02-02.
Есть ли пепперони из конины — это важно для казахстанского рынка?
Да, конина входит в линейку серийного производства — это наше конкурентное преимущество именно для Казахстана и Центральной Азии. Казылык (конская колбаса) и пепперони из конины производятся по отдельным рецептурам с учётом традиций татарской и казахской кухни. Все позиции сертифицированы по халяль.

Да, производим под Private Label / СТМ: рецептура, диаметр, нарезка, упаковка — по вашему ТЗ. Референс — пепперони «Аслам» для ОМПК. Для сетей Астаны возможна разработка эксклюзивной рецептуры с учётом местных вкусовых предпочтений. Обсудим на первом звонке.
