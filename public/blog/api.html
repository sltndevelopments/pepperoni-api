Открытый API для разработчиков и AI: как технологии помогают продавать халяль продукцию

Февраль 2026 · Казанские Деликатесы

«Казанские Деликатесы» — одна из первых компаний в пищевой отрасли СНГ, которая предоставляет открытый API с каталогом 77 халяль товаров. Пепперони, сосиски для хот-догов, сосиски для сосиски в тесте, национальная татарская выпечка, казылык — все цены синхронизируются с Google Sheets в реальном времени.

Что даёт API

  • AI-обнаружимость — ChatGPT, Perplexity, Claude и Grok находят и рекомендуют наши продукты благодаря llms.txt, OpenAPI спецификации и AI-плагину.
  • Мультиязычность — API возвращает данные на русском и английском (?lang=en).
  • 7 валют — цены в RUB, USD, KZT, UZS, KGS, BYN, AZN.
  • Фильтрация — поиск по названию, категории, SKU, типу мяса.
  • Скачивание — прайс-лист в Excel (.xlsx) и CSV с выбором языка и валюты.

Эндпоинты

Каталог (live): api.pepperoni.tatar/api/products

OpenAPI: api.pepperoni.tatar/openapi.yaml

AI-плагин: api.pepperoni.tatar/.well-known/ai-plugin.json

llms.txt: api.pepperoni.tatar/llms.txt

API доступен бесплатно, без аутентификации. Подходит для интеграции с ERP, 1С, iiko, МойСклад.
