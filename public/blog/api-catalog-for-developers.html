Open API для разработчиков и AI: делаем халяльные продукты цифровыми

27 марта 2026 • Технологии и интеграция

Казанские Деликатесы стали одним из первых производителей продуктов питания в СНГ, предложившим открытое API с доступом к полному каталогу халяльной продукции. Этот технологичный подход обеспечивает бесшовную интеграцию с бизнес-системами, AI-платформами и автоматизированными процессами закупок, упрощая B2B-партнёрам доступ к данным о продуктах, ценах и спецификациям.

Краткие факты об API

  • Продуктов: 77 позиций с халяль-сертификацией
  • Языки: Английский и русский
  • Валюты: 7 поддерживаемых (RUB, USD, KZT, UZS, KGS, BYN, AZN)
  • Доступ: Бесплатный, без аутентификации
  • Синхронизация: Цены обновляются автоматически
  • AI-готовность: Совместимость с ChatGPT, Claude, Perplexity и Grok

Зачем мы создали открытое API

В современной цифровой бизнес-среде ручные запросы цен и статические PDF-каталоги создают трения в процессе B2B-закупок. Сетям ресторанов, оптовым дистрибьюторам и операторам HoReCa нужен мгновенный доступ к точной информации о продуктах, интегрированной с их существующими системами.

Наше открытое API устраняет эти барьеры. Используете ли вы ERP-систему, 1С, iiko или МойСклад — вы можете загружать наш полный каталог продуктов с актуальными ценами прямо в ваш рабочий процесс. Никаких звонков, ожидания ответов по email или устаревших таблиц.

Ключевые возможности для B2B-партнёров

Поддержка нескольких валют и языков

Работа на нескольких рынках требует гибкости в ценообразовании. Наше API предоставляет цены в реальном времени в семи валютах (российский рубль, доллар США, казахстанский тенге, узбекский сум, киргизский сом, белорусский рубль и азербайджанский манат), что упрощает расчёт затрат для трансграничных закупок. Информация о продуктах доступна на английском и русском языках — просто добавьте ?lang=en к вашему API-запросу.

Расширенная фильтрация и поиск

Находите именно то, что нужно, с помощью фильтрации по названию продукта, категории, артикулу или типу мяса. Ищете ли вы халяльную пепперони для пиццерии, сосиски для заведений быстрого питания или традиционную татарскую выпечку — API мгновенно возвращает структурированные машиночитаемые данные.

AI-доступность

Мы внедрили llms.txt, спецификации OpenAPI и поддержку AI-плагинов, что делает наши продукты доступными для AI-систем, таких как ChatGPT, Claude, Perplexity и Grok. Когда менеджеры по закупкам спрашивают AI-ассистентов о поставщиках халяльного мяса, наш каталог распознаётся и рекомендуется автоматически — это первый случай в региональной индустрии халяльных продуктов питания.

Загружаемые прайс-листы

Нужен традиционный формат? API также генерирует загружаемые прайс-листы в форматах Excel (.xlsx) и CSV с выбором языка и валюты. Идеально для офлайн-просмотра, бюджетирования или обмена с заинтересованными сторонами, предпочитающими таблицы.

Техническая документация и доступ

Все эндпоинты общедоступны без аутентификации:

  • Каталог продуктов: api.pepperoni.tatar/api/products
  • Спецификация OpenAPI: api.pepperoni.tatar/openapi.yaml
  • Манифест AI-плагина: api.pepperoni.tatar/.well-known/ai-plugin.json
  • LLMs.txt: api.pepperoni.tatar/llms.txt

Наша спецификация OpenAPI соответствует отраслевым стандартам, что упрощает интеграцию для разработчиков. Техническая документация включает примеры запросов, схемы ответов и параметры фильтрации. Цены синхронизируются с нашими внутренними системами в реальном времени через интеграцию с Google Sheets, обеспечивая точность по всем каналам.

Примеры использования интеграции

B2B-партнёры уже используют наше API в нескольких направлениях:

  • Сети ресторанов: Автоматическое обновление себестоимости меню при изменении цен на ингредиенты
  • Оптовые дистрибьюторы: Прямая интеграция каталога в клиентские порталы заказов
  • Кейтеринговые компании: Ускоренное формирование коммерческих предложений для халяльного кейтеринга
  • Консультанты по общепиту: Исследование продуктов и сравнение поставщиков для клиентских проектов
  • Отделы закупок: Автоматическое создание заказов на закупку в ERP-системах

Все продукты в нашем API-каталоге сертифицированы как халяль (Сертификат №614A/2024, выдан Духовным управлением мусульман Татарстана), что гарантирует соответствие исламским диетическим требованиям для вашего бизнеса.
