# Единая линия компании: кризисный регламент

> Внутренний документ. Раздать руководителям отделов. Задача: вся компания действует и говорит одинаково. Утвердить приказом руководителя.

## Принцип №1: один голос

**Публично о ситуации говорит ТОЛЬКО уполномоченный спикер.** Все остальные сотрудники — переадресуют. Это не про секретность, это про точность: разноголосица = повод для новой атаки.

| Канал обращения | Кто отвечает |
|---|---|
| СМИ, журналисты, Telegram-каналы | Только спикер / PR (см. ниже) |
| Регулятор (РПН, РСХН) | Только руководитель + юрист |
| Клиенты, партнёры, ритейл | Коммерческий директор по утверждённому скрипту |
| Соцсети, комментарии | SMM по утверждённому скрипту, без импровизации |
| Сотрудники между собой / семьям | По внутренней памятке (раздел 6) |

## Принцип №2: ничего не публикуем сырым

Урок ронидазола: **открытые данные использовали против вас.** Поэтому:

- Наружу — только **финальные документы** (госпротоколы, заключения), не черновики и не внутренняя переписка.
- Любая публикация — после визы юриста.
- Не выкладываем персональные данные сотрудников, поставщиков, внутренние номера партий без необходимости.

## Алгоритм при новом инфоповоде (первые 60 минут)

1. **Зафиксировать** (скрин, ссылка, время, источник) — не реагировать сразу.
2. **Сообщить в штаб** — не отвечать самостоятельно.
3. **Юрист + качество** за 30 минут готовят фактуру: что правда, что нет.
4. **Спикер** даёт один выверенный ответ. Если просят комментарий с дедлайном — дать хотя бы «готовим официальную позицию, предоставим до [время]» (история Mash: не успели ответить до публикации — больше так не делаем).
5. **Единый текст** рассылается всем каналам одновременно.

## Единые тезисы (для ВСЕХ, заучить)

Эти 4 фразы — каркас. Никто не добавляет от себя:

1. «Нашу продукцию **проверил Роспотребнадзор** — она **соответствует** требованиям безопасности» (протокол № 16-01/30124-26 от 05.06.2026).
2. «**Экспресс-тест — это скрининг, а не диагноз.** Точный результат даёт только лабораторный посев».
3. «Наша котлета — **полуфабрикат, который обязательно жарят.** Безопасность готового блюда зависит от приготовления».
4. «У нас **полная прослеживаемость** сырья через "Меркурий" и добровольный контроль по 65+ показателям».

## Чего НЕ делать (стоп-лист)

- Не спорить и не хамить в комментариях.
- Не нападать на регулятора — только корректные процессуальные формулировки.
- Не обещать того, чего не сделали («все пробы чистые», пока нет протокола).
- Не пересылать внутренние документы в личных переписках.
- Не комментировать «по горячему» без визы штаба.

## Памятка рядовому сотруднику

Если спрашивают (журналист, знакомый, в соцсетях):

> «Я не уполномочен(а) комментировать. Официальную позицию даёт пресс-служба, вот контакт: [___]».

И сообщить в штаб. Всё. Никакой импровизации — даже из лучших побуждений.

## Системные меры (чтобы не повторялось)

1. **Ввести анализ готовых котлет/полуфабрикатов на сальмонеллу** в квартальный мониторинг (сейчас тестируется только сырьё — это была дыра).
2. **Устранить замечания предписания досрочно** (до 10.08.2026) с фотоотчётом.
3. **Завершить FSSC 22000** — сильный аргумент в любом споре.
4. **Условия отгрузки на выездные мероприятия:** прикладывать к партии памятку «полуфабрикат, требует термообработки ≥75 °C», фиксировать передачу актом — чтобы зона ответственности была документально разграничена.
5. **Реестр инцидентов:** вести лог (дата, повод, что сделали) — пригодится при системной атаке.
