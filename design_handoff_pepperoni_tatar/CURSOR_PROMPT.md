# Промпт для Cursor

Ниже — готовый шаблон промпта, который можно вставить в Cursor (или в любого AI-агента-разработчика) для применения дизайна к существующему сайту pepperoni.tatar.

**Перед вставкой:**

1. Положите всю папку `design_handoff_pepperoni_tatar/` в корень своего репо (или в `docs/`)
2. Откройте Cursor в репо
3. Выберите ОДНО направление: `editorial`, `manifest`, `ornament` или `catalog`
4. Вставьте промпт ниже, заменив `{{DIRECTION}}` на выбранное и `{{LANG}}` на язык по умолчанию (`tt` / `ru` / `en`)

---

## Промпт (скопировать целиком)

```
Контекст
========
Это сайт pepperoni.tatar — B2B халяль-мясокомбинат из Татарстана.
Структура сайта и контент остаются прежними. Меняем только визуальный язык
по дизайн-пакету в /design_handoff_pepperoni_tatar/.

Прочти сначала:
- design_handoff_pepperoni_tatar/README.md       — общая логика дизайна
- design_handoff_pepperoni_tatar/tokens.css      — все CSS-переменные
- design_handoff_pepperoni_tatar/preview/v{{DIRECTION}}.jsx   — выбранное направление как референс
- design_handoff_pepperoni_tatar/preview/shared.jsx — i18n-словарь, орнаменты

Что сделать
===========
1. Подключи tokens.css в основной шаблон сайта (до всех остальных стилей).
   Скопируй файл в стандартное место CSS-ассетов проекта; путь подбери под
   текущую структуру.

2. Установи на <html> атрибут data-direction="{{DIRECTION}}" — это
   активирует нужную палитру и типографику из tokens.css.

3. Перевёрстай существующие секции лендинга под визуальный язык
   {{DIRECTION}}-варианта, используя ТОЛЬКО токены из tokens.css
   (`var(--pt-bg)`, `var(--pt-fg)`, `var(--pt-accent)`, `var(--pt-text-3xl)` и т.д.).
   Никаких хардкод-цветов / шрифтов / отступов.

4. Контент НЕ меняй — оставь существующие тексты, описания продуктов,
   контакты, формы. Если для какой-то секции в дизайне нет аналога —
   адаптируй её к новому стилю, не выкидывай.

5. Орнаменты (StarRub, Tulip, DiamondStrip, HalalSeal) — собери как inline
   SVG-компоненты в стеке проекта. См. реализацию в shared.jsx, секция
   "Ornaments". Они построены ТОЛЬКО из примитивов (rect / circle),
   повторение тривиально.

6. Шрифты подгружаются через @import в tokens.css. Если на сайте уже есть
   font-loader / next/font / similar — перенеси импорт туда для
   производительности.

7. Мультиязычность (TT/RU/EN). Если на сайте уже есть i18n — оставь как
   есть, дизайн с этим работает. Если ещё нет — посмотри в shared.jsx
   объект `T`, там словарь всех ключевых строк сайта на трёх языках.

8. Плейсхолдеры под фотографии (полосатые блоки с подписью "product shot")
   ОСТАВЬ — продакшен-фото будут предоставлены отдельно. Размер блока
   зафиксируй из дизайна.

9. Респонсив: сейчас макеты сделаны под 1440px desktop. Для брейкпоинтов
   ниже 1024px — двухколоночные сетки в одну, hero-типографика
   уменьшается до 56-72px, padding с 80px до 24px. Логика стандартная,
   действуй по уму.

Чего НЕ делать
==============
- Не переписывай контент / тексты / описания продуктов
- Не меняй структуру URL / навигации / форм
- Не добавляй новых страниц без явного запроса
- Не используй цвета / шрифты / размеры вне токенов
- Не пытайся "улучшить" дизайн — это финальный, согласованный макет.
  Если кажется что что-то не так — спроси, не правь молча.

Проверка
========
После всех изменений:
- Открой каждую секцию и сравни глазами с preview/index.html
  (открыть локально, выбрать в Tweaks нужное направление)
- Убедись что lang-переключатель TT/RU/EN продолжает работать
- Проверь шрифты загружаются (Geist, Instrument Serif, Geist Mono,
  Noto Naskh Arabic)
- Проверь что на data-direction="editorial" / "manifest" / "ornament" /
  "catalog" сайт переключается между палитрами одной строкой

Действуй пошагово. Делай маленькие коммиты по одной секции. После каждой
секции показывай результат до того, как двигаться дальше.
```

## Если хотите смикшировать направления

Например, типографика от Editorial + орнаменты от Ornament + плотность от Catalog:

```
Вместо целиком одного направления возьми элементы из нескольких:
- Типографика и общая ритмика — из v1-editorial.jsx
- Орнаменты (звезда, тюльпан, печать HALAL, бордюры) — из v3-ornament.jsx
  Расставляй сдержанно: один-два акцента на секцию, не как фон.
- Плотность сетки продукции — из v4-catalog.jsx (4×6 = 24 SKU видно сразу)
- Палитра: data-direction="editorial" (крем как база, изумруд+золото как
  акценты)

Остальные пункты промпта те же.
```

Скажите мне, что именно — я уточню дизайн или вышлю более конкретные спеки.
Дизайн-токены — единственный источник правды; меняем их здесь, синхронизируем
в репо одним файлом.
