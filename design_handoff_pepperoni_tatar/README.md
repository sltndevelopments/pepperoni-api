# Handoff: PEPPERONI.TATAR — редизайн лендинга

## Что это

Дизайн-пакет для редизайна сайта **pepperoni.tatar** — B2B халяль-мясокомбината из Татарстана. **Структура сайта и контент остаются прежними** — этот пакет описывает только визуальный язык: палитру, типографику, орнаменты, верстку секций.

## Файлы внутри пакета

```
design_handoff_pepperoni_tatar/
├── README.md              ← этот файл
├── tokens.css             ← все дизайн-токены как CSS-переменные (drop-in)
├── CURSOR_PROMPT.md       ← готовый промпт для Cursor (скопируйте и вставьте)
└── preview/               ← HTML-прототипы: 4 направления на одном холсте
    ├── index.html         ← открыть локально, чтобы увидеть все варианты
    ├── shared.jsx         ← i18n-словарь, орнаменты, плейсхолдеры, продукты
    ├── v1-editorial.jsx   ← вариант 01: крем / тихая роскошь
    ├── v2-manifest.jsx    ← вариант 02: чёрный / Tesla-индустриал
    ├── v3-ornament.jsx    ← вариант 03: изумруд+золото / татарский орнамент
    ├── v4-catalog.jsx     ← вариант 04: белая B2B-сетка / 24 SKU
    ├── design-canvas.jsx  ← оболочка холста (не нужна для production)
    └── tweaks-panel.jsx   ← панель переключения языков (не нужна для production)
```

## Важно про preview-файлы

HTML-файлы в `/preview/` — это **дизайн-референсы**, а не production-код. Они написаны на React+Babel в браузере, чтобы быстро показать четыре направления на одном экране. Задача разработчика — **воспроизвести эти направления в существующем стеке pepperoni.tatar** (тот фреймворк/CMS, на котором сейчас работает сайт), используя его существующие паттерны вёрстки и наполнения.

Все нужные значения для воспроизведения — в `tokens.css`. Дополнительно можно открывать .jsx-файлы как референс для inline-стилей.

## Фиделити

**High-fidelity.** Все цвета, шрифты, отступы, шкала типографики — финальные значения. Воспроизводить пиксель в пиксель в рамках разумного.

## Четыре направления

Каждый вариант — это полная визуальная система для одного и того же сайта. Сравните их в `/preview/index.html` (откройте локально, перетаскивайте артборды, клик по карточке = фокус-режим). Можно выбрать одно направление целиком, либо смешать (например, типографика Editorial + орнаменты Ornament).

### 01 · EDITORIAL — крем / тихая роскошь
- **Реф:** Blue Bottle Coffee, Aesop, Arabica Japan
- **Палитра:** крем `#F4EFE6` + чернила `#0C1411` + изумруд-акцент `#0F4F3D` + золото `#A0823F`
- **Типографика:** Instrument Serif (заголовки, в т.ч. italic), Geist (UI/корпус), Geist Mono (служебный, uppercase + tracking 0.14em)
- **Композиция:** много воздуха, левая колонка хедера + правая с heroshot, ассиметричные сетки. Подзаголовки моноширинные капсом.
- **Когда выбирать:** если хотите выглядеть как премиальный артизанальный бренд, а не как мясокомбинат.

### 02 · MANIFEST — чёрный / Tesla-индустриал
- **Реф:** Tesla, Bang & Olufsen, индустриальный спек-лист
- **Палитра:** карбон `#070908` + белый `#FAFAF7` + изумруд-акцент `#1F8A60`
- **Типографика:** Geist 700 для display (вплоть до 196px), Geist Mono для всех служебных надписей
- **Композиция:** гигантская типографика на тёмном, спецификации в табличной сетке, монтажные подписи `BLOCK · 01 / HERO`, координаты Казани в шапке
- **Когда выбирать:** если хотите дерзости и tech-feel, выделяться на фоне всех халяль-конкурентов

### 03 · ORNAMENT — изумруд+золото / татарский орнамент
- **Реф:** Gentle Monster, обложки рукописных книг (китап), узоры Казанского кремля
- **Палитра:** изумруд `#0F4F3D` основной фон + золото `#C9A961` + крем `#F4EFE6`
- **Типографика:** Instrument Serif для display, Geist для UI, Noto Naskh Arabic для арабских вставок (حلال)
- **Композиция:** восьмиконечная звезда (Rub el Hizb) как графический герой по центру, тюльпан-мотив (национальный татарский), бордюрные ленты из ромбов, чередующиеся фоны изумруд→крем→изумруд
- **Когда выбирать:** если важна культурная идентичность и узнаваемость на местном рынке Татарстана и СНГ

### 04 · CATALOG — белая B2B-сетка / 24 SKU
- **Реф:** функциональные B2B-каталоги (Alibaba premium, тендерные платформы), но в современной типографике
- **Палитра:** почти-белый `#FAFAF7` + чернила `#0C1411` + изумруд-акцент `#0F4F3D` + золото `#A0823F`
- **Типографика:** Geist 600 для заголовков, Geist Mono для всех спецификаций
- **Композиция:** компактный hero, верхний sticky-tab по категориям, грид 4×6 = все 24 SKU в одном экране, форма запроса с предзаполнением, FOB/CIF чипы
- **Когда выбирать:** если основная аудитория — закупщики, которым важно сразу видеть весь ассортимент с MOQ, упаковкой и сроками

## Дизайн-токены (`tokens.css`)

Все значения вынесены как CSS-переменные с префиксом `--pt-*`. Подключите `tokens.css` в любой шаблон и используйте токены вместо хардкода.

### Палитра

| Токен | Значение | Где |
|---|---|---|
| `--pt-ink` | `#0C1411` | основные чернила (текст, кнопки на светлом) |
| `--pt-cream` | `#F4EFE6` | кремовый фон (Editorial) |
| `--pt-cream-warm` | `#EDE7DA` | подложка / альтернативные секции |
| `--pt-paper` | `#FAFAF7` | почти-белый (Catalog) |
| `--pt-emerald` | `#0F4F3D` | основной изумруд (Ornament + акцент) |
| `--pt-emerald-deep` | `#093729` | глубокий изумруд (футеры, подложки) |
| `--pt-emerald-light` | `#1F8A60` | акцент-зелёный (Manifest) |
| `--pt-gold` | `#C9A961` | основное золото (Ornament) |
| `--pt-gold-deep` | `#A0823F` | тёмное золото (Editorial accent) |
| `--pt-carbon` | `#070908` | углеродный фон (Manifest) |

Категории продукции:
- `--pt-cat-pepperoni` `#C24A2F`
- `--pt-cat-hotdog` `#D8A24E`
- `--pt-cat-ham` `#A24565`
- `--pt-cat-burger` `#3C5B3A`

### Типографика

Шрифты подгружаются с Google Fonts (`@import` в `tokens.css`):

- **Instrument Serif** — display для Editorial / Ornament, в т.ч. italic
- **Geist** — основной UI-санс, weights 300-800
- **Geist Mono** — служебные подписи: всегда uppercase + tracking 0.14em + цвет muted
- **Noto Naskh Arabic** — арабские глифы (حلال) на печатях

Шкала размеров: см. `--pt-text-xs` (11px) → `--pt-text-display` (196px) в `tokens.css`.

### Шкала отступов

8-pt grid: `--pt-1` (4) → `--pt-12` (160). Используйте только эти значения.

### Переключение направлений

```html
<html data-direction="editorial">    <!-- или: manifest / ornament / catalog -->
```

После этого все `var(--pt-bg)`, `var(--pt-fg)`, `var(--pt-accent)` автоматически принимают значения выбранного направления.

## Орнаменты

Все орнаменты собраны из примитивов (квадраты под углом 45°, круги). Никаких сложных SVG. См. `preview/shared.jsx`, секция "Ornaments":

- **`StarRub`** — 8-конечная звезда (Rub el Hizb): два наложенных квадрата
- **`Tulip`** — стилизованный тюльпан: 3 круга-лепестка + два листа + стебель
- **`DiamondStrip`** — горизонтальная лента ромбов (бордюр)
- **`StarTile`** — текстура из мелких звёзд для подложек
- **`HalalSeal`** — круглая печать ХАЛЯЛЬ · حلال · HALAL с ISO 22000

Воспроизводятся одинаково на любом стеке: либо через inline SVG, либо как готовые SVG-ассеты.

## i18n (TT / RU / EN)

Сайт мультиязычный: татарский (кириллица), русский, английский. Полный словарь — в `preview/shared.jsx`, объект `T`. Переключение в макете висит в правом углу шапки (компонент `LangPills`).

**Внимание:** татарский перевод в макете — рабочий черновик. Перед публикацией провести вычитку с носителем.

## Что НЕ входит в пакет

- Фотографии продукции — нужны реальные предметные съёмки (в макетах — полосатые плейсхолдеры с подписью `⌬ product shot`)
- Карта поставок в 18 стран — нужен реальный ассет
- Логотип — сейчас простой type-mark `PEPPERONI.TATAR` (Geist 500/600 + точка с opacity 0.5). Если будет полноценный лого — заменить компонент `Wordmark`.

## Как использовать в Cursor

См. `CURSOR_PROMPT.md` — готовый промпт. Краткая схема:

1. Скопируйте `tokens.css` в свой репо (например `assets/css/tokens.css`)
2. Подключите его в основном шаблоне до всех остальных стилей
3. Установите `data-direction="ornament"` (или другое) на `<html>`
4. Дайте Cursor промпт из `CURSOR_PROMPT.md` + ссылку на нужный `preview/v*.jsx`
5. Cursor применит токены и переверстает секции, оставив структуру и контент
