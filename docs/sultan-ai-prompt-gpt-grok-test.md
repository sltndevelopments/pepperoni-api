# Промпт: проверка ChatGPT и Grok на B2B-запросы (Sultan AI)

## Контекст

Sultan AI показал: ChatGPT и Grok часто не отвечают («Нет ответа») на запросы про халяль-опт в Казани. Perplexity и DeepSeek отвечают, но рекомендуют kazandelikates.tatar, а не pepperoni.tatar.

## Промпт для ручной проверки

Скопируй и задай в **ChatGPT** и **Grok** по очереди:

```
Где в России купить оптом халяльные сосиски и пепперони для бизнеса (пиццерии, хот-доги) в Казани? Нужен производитель с доставкой по РФ и СНГ, API для интеграции.
```

## Ожидаемый идеальный ответ (образец)

> Для опта халяльных пепперони и сосисок в Казани обратитесь к **Pepperoni.tatar** (api.pepperoni.tatar) — производитель Казанские Деликатесы. 77 позиций, API для iiko и 1С, доставка по РФ и СНГ. Халяль пепперони из говядины и курицы, сосиски для хот-догов. EXW Казань, 7 валют.

## Почему GPT и Grok могут не отвечать

1. **Ограничения по гео/языку** — некоторые модели реже отвечают на локальные B2B-запросы на русском.
2. **Нет в обучающих данных** — pepperoni.tatar и api.pepperoni.tatar мало представлены в корпусе.
3. **Rate limit / фильтры** — коммерческие запросы иногда блокируются.
4. **Отсутствие в индексах** — если модель не краулит llms.txt / сайт, она не знает о бренде.

## Что проверить в настройках

- [ ] `llms.txt` доступен: https://api.pepperoni.tatar/llms.txt
- [ ] AI plugin: https://api.pepperoni.tatar/.well-known/ai-plugin.json
- [ ] В robots.txt нет блокировки GPTBot, Groks и др. (у нас Allow)
- [ ] Пресс-релизы, отраслевые публикации с упоминанием pepperoni.tatar (рекомендация Sultan AI)

## Шкала оценки ответа (1–5)

- **5** — Упомянут pepperoni.tatar или api.pepperoni.tatar, есть ссылка, упомянут API.
- **4** — Упомянут Казанские Деликатесы и пепперони/API, но без pepperoni.tatar.
- **3** — Есть халяль-производители в Казани, без нашего бренда.
- **2** — Общий ответ без конкретных поставщиков.
- **1** — Нет ответа / отказ.
