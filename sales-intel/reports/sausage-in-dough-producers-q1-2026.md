## Ограничения методики

1. **Сайты на JS-рендеринге** (React/Next.js без SSR) отдают почти пустой HTML — мы ловим 0 упоминаний. Это ≈ 15–20 компаний из "no".
2. **Каталог в PDF / картинках** — текст не индексируется HTML-парсером. Типично для старых хлебокомбинатов.
3. **Региональные сетевые пекарни** (Бегемаг, Булкин, Хлебный дом) могут продавать через сеть «наша кухня» без публичного каталога SKU — мы их поймали только по Бегемагу, остальные требуют точечной проверки.
4. **B2B-производители для HoReCa/ритейла** (ООО «Альтернатива» / BONTIER) часто прячут полный ассортимент за «запросите у менеджера», и тогда на сайте есть только маркетинговое упоминание SKU.

### Что ещё можно сделать

- Поднять порог `--top` до 300 (+150 компаний 300-700 млн ₽).
- Для Tier 4 (без сайта) — прогнать Яндекс.Поиск `<INN> официальный сайт` или 2ГИС Places API.
- Для Tier 2 → сделать ручной обзвон «есть ли сосиски в тесте в продуктовой линейке».
- Для Tier 1 → сразу отправить коммерческое: «у нас халяль-сосиски для сосиски-в-тесте от X ₽/кг, лучше вашего текущего поставщика на Y%».
