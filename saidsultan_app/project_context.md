# Генеральное Техническое Задание (ТЗ) для стартапа "AI Visibility SaaS"

## 1. Бизнес-контекст
Мы разрабатываем B2B SaaS-платформу для управления видимостью брендов в ИИ-поисковиках (ChatGPT, Perplexity, Gemini). 
Наш MVP (минимально жизнеспособный продукт) строится на домене `saidsultan.com`.
Наш первый "нулевой" клиент — компания "Казанские Деликатесы" (производитель халяльного мяса). 

## 2. Инфраструктура
- Сервер: Ubuntu (IP 37.9.4.101).
- Основной домен приложения: saidsultan.com (проксируется через Cloudflare).
- Дополнительные ресурсы: сетка из 25+ доменов в Cloudflare, на которые идет трафик ИИ-ботов. Мы будем использовать их как PBN (Private Blog Network) для обучения нейросетей.

## 3. Технологический стек
- Язык: Python 3
- Фреймворк: FastAPI (для быстрого API и веб-интерфейса)
- Сервер: Uvicorn
- Взаимодействие с ИИ: OpenAI API
- Интеграция: Cloudflare API (для управления доменами и анализа логов ботов)

## 4. Архитектура MVP (Модули)
Проект должен состоять из следующих логических блоков:
1. `core/scanner.py` — модуль отправки промптов в LLM и сбора ответов.
2. `core/advisor.py` — RAG-система (на основе файла с досье компании), которая анализирует ответы ИИ и дает рекомендации.
3. `services/cloudflare.py` — модуль для работы с API Cloudflare (получение логов, управление 301 редиректами для нашей сетки доменов).
4. `api/routes.py` — эндпоинты FastAPI для управления системой.

## 5. Требования по безопасности (Критично!)
Поскольку мы работаем с пулом доменов, которые атакуют боты:
1. Запрет прямого доступа по IP: FastAPI должен обрабатывать только те запросы, которые прошли через Cloudflare.
2. Middleware безопасности: Необходимо проверять наличие заголовков Cloudflare (например, `CF-Connecting-IP`, `True-Client-IP`).
3. Скрытые ключи: Все API ключи (OpenAI, Cloudflare) должны строго лежать в файле `.env` и загружаться через `pydantic-settings` или `python-dotenv`.

## 6. Задача для Cursor Agent:
Опираясь на этот контекст, сделай следующее:
1. Инициализируй проект: создай `requirements.txt` с нужными библиотеками.
2. Создай структуру папок (`core`, `api`, `services`, `data`).
3. Напиши `main.py` с инициализацией FastAPI и строгим Middleware для проверки Cloudflare-заголовков.
4. Выведи на главную страницу (GET "/") сообщение: `{"status": "Saidsultan AI Visibility Core is running securely"}`.
