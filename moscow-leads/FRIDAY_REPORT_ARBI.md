# Пятничный отчёт Арби (ручной, до включения автодайджеста)

Отправляется каждую пятницу до 17:00 текстом (не голосовым). Даже в тихую неделю — с нулями.

Когда сданы Задачи 1–4 бота (`scheduler.py digest`), этот ручной отчёт отменяется:
бот шлёт тот же смысл сам владельцу и Арби.

```
ОТЧЁТ ЗА НЕДЕЛЮ [даты]

1. ЛИДЫ
Пришло новых: ___
Обработано (связался): ___
В работе сейчас: ___

2. ДВИЖЕНИЕ
Отправлено образцов: ___ (кому: ...)
Проведено встреч/дегустаций: ___ (с кем: ...)
Передано дистрибьюторам: ___ (кому и какому дистру: ...)

3. РЕЗУЛЬТАТ
Первых отгрузок: ___ (клиент / объём)
Повторных отгрузок: ___ (клиент / объём)

4. ЗАСТРЯЛО (>72ч без ответа)
- [клиент] — передан [дистр] [дата] — обратной связи нет
- ...

5. БЛОКЕРЫ (что нужно от Рината)
- ...
- Если ничего не нужно — пишем «нет»

6. ЭКСПОРТ (отдельной строкой)
Азербайджан: [статус одной строкой]
Прочее: [статус или «без движения»]
```

Правила: факты только текстом. Пункт 4 — главный (лиды не должны исчезать у дистрибьюторов).
